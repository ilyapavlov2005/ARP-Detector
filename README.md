ARP Spoofing Detector
Обнаружение ARP-атак в локальной сети

О проекте
ARP Spoofing Detector — это кроссплатформенное графическое приложение на Java для обнаружения ARP-атак (ARP-spoofing) в локальных сетях. Программа в реальном времени мониторит ARP-таблицу операционной системы, выявляет подозрительную активность и оповещает пользователя об атаках.

Основные возможности
Функция	Описание
Мониторинг ARP-таблицы	Сканирование ARP-таблицы каждые 3 секунды
Обнаружение атак	Выявление подмены шлюза, изменения MAC-адреса, ARP-флуда
Список устройств	Отображение всех устройств в сети с IP, MAC, производителем и статусом
Фильтрация и поиск	Поиск по IP, MAC или производителю + фильтр по статусу
Оповещения	Цветовая индикация угроз, звуковой сигнал, мигание окна
Статистика	ASCII-графики распределения угроз и активности по часам
Экспорт отчётов	Сохранение отчёта в текстовый файл
Активное сканирование	Ping sweep для обнаружения всех устройств в сети
Типы обнаруживаемых атак
Тип атаки	Уровень угрозы	Описание
Подмена шлюза	Критический	Злоумышленник выдаёт себя за роутер
Изменение MAC-адреса	 Средний	У устройства изменился MAC-адрес
ARP-флуд	 Средний	Слишком много ARP-пакетов в секунду (DoS)
Быстрый старт

Установка
Способ 1: Портативная версия (рекомендуется)
Скачай последний релиз с Releases

Распакуй ZIP-архив

Запусти run.bat от имени администратора

# Компиляция
javac -d out src/com/arpdetector/*.java

# Запуск
java -cp out com.arpdetector.ARPDetectorApp

Важно! Программа должна запускаться от имени администратора (Windows) или с правами root (Linux/macOS) для доступа к ARP-таблице.

# Структура проекта
src/com/arpdetector/

├── ARPDetectorApp.java        # Точка входа

├── MainWindow.java            # Главное окно (GUI)

├── ARPSpoofDetector.java      # Ядро — обнаружение атак

├── DeviceTableModel.java      # Модель таблицы устройств

├── AlertListModel.java        # Модель списка оповещений

├── Device.java                # Модель устройства

├── Alert.java                 # Модель оповещения

└── MacVendorLookup.java       # Определение производителя по MAC

🔧 Технологии
Технология	Назначение

AWT	Цвета, компоновка, события
